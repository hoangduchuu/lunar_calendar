# Tech Stack

- Flutter (Dart SDK >=3.0.6 <4.0.0)
- UI: Material, Google Fonts
- Localization: flutter_localizations, intl, ARB files in `lib/l10n`
- State/Navigation: Flutter `MaterialApp` routes with constants in `lib/router.dart`
- Data storage: sqflite (SQLite), path
- Notifications: flutter_local_notifications, flutter_native_timezone, flutter_timezone
- Gestures/UI: simple_gesture_detector, sliding_up_panel
- Platforms: Android, iOS, Web, macOS, Windows, Linux
- Linting: `flutter_lints` via `analysis_options.yaml`
